# BPMNCheckingSoundness
Checking soundness of BPMN models using Causal Footprint (base on ProM 5.2)
#Building
1. Create a new Java 1.7 source project in Eclipse.
2. Add the "src" folder as source
3. Add the "images" folder as resources
4. Add the "lib" folder as library
5. Add all .ini file to project
#Run Project
Select Run As \Run Configurations, on Main tab, at Main Class enter: org.processmining.ProM
